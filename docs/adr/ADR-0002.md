# ADR-0002 · Motor de reglas como funciones puras en el dominio

| Campo          | Valor                                                              |
|----------------|--------------------------------------------------------------------|
| Estado         | Aceptada · 26-06-2026                                             |
| Contexto       | Se exige una suite de tests que valide el camino feliz y un caso de error de la funcionalidad crítica. El jurado puede preguntar por cualquier parte del código. |
| Decisión       | Implementar cada bandera como función pura, sin dependencias de framework ni DB, dentro de `domain/`. |
| Consecuencias  | **+** Testeable y explicable en el Q&A.<br>**+** Reutilizable en ETL y API.<br>**−** Requiere mapear con cuidado los campos de OCDS. |
